# The Rapid Reservoir-River Temperature Assessment Model (R3TAM)

This is a collection of documents that describe the components of the Rapid Resevoir-River Temperature Assessment Model (R3TAM) framework and present examples of its usage in the Shasta-Keswick-Sacramento River system in California.

The R3TAM framework is flexible and may evolve as it is tested in various applications.
